# Radio Button

Radio Buttons ermöglichen eine Auswahl aus einer Gruppe sich gegenseitig ausschließender Optionen. Wenn ein:e Nutzer:in eine andere Option wählt, wird die vorherige Auswahl automatisch deselektiert.

Es können immer nur mehrere Radio Buttons als Gruppe eingesetzt werden. Bei einer großen Anzahl an Optionen und begrenztem Platz kann auch ein [Select](?path=/docs/components-input-select) eingesetzt werden.

Radio Buttons **werden verwendet**:

- in Formularen, z.B. Anrede
- für die Auswahl von Filteroptionen

**Nicht verwendet** werden Radio Buttons für eine Mehrfachauswahl. Stattdessen kommen [Checkboxes](?path=/docs/components-input-checkbox) zum Einsatz.

## Anatomie

Ein Radio Button besteht aus dem eigentlichen Eingabefeld und einem Label. Optional kann eine Gruppe mehrerer Radio Buttons ein Label erhalten.

<!-- Optional: schematische Darstellung -->

1. **Group label** (optional): Beschreibt den Inhalt der Gruppe
2. **Input**: Eingabefeld des Radio Buttons, zeigt den Zustand (_unselected_, _selected_, _indeterminate_) an
3. **Radio Button Label**: Beschreibt, was der:die Nutzer:in auswählt
