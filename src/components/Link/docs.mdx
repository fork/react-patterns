# Link

Mit Links können Nutzer:innen navigieren. Links können eigenständig oder im Text verwendet werden.

Links **werden verwendet**:

- für die Navigation zu anderen Seiten innerhalb der Anwendung/Website
- für den Zugriff auf Ressourcen innerhalb der Anwendung/Website (z.B. Downloads)
- für die Navigation zu externen Websites bzw. Ressourcen
- als Sprungmarken innerhalb einer Seite
- um Telefonnummern oder Emailadressen nutzbar zu machen

**Nicht verwendet** werden Links, um Aktionen auszulösen, z.B. Daten in der Anwendung oder die Art der Darstellung zu ändern. Für Aktionen sollten [Buttons](?path=/docs/components-input-button) genutzt werden.
