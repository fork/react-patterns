# Richtext

Die Richtext-Komponente wird für die Darstellung von formatiertem Text verwendet.

Folgende Formatierungsoptionen werden unterstützt:

- Überschriften der Ebenen 2 und 3
- Auszeichnungen: bold und italic
- Inline-Links
- Listen
  - geordnete Listen
  - ungeordnete Listen
