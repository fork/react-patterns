# Button

Buttons sind klickbare Elementen, mit denen Aktionen ausgelöst werden.

Buttons **werden verwendet**, wenn ein:e Nutzer:in:

- ein Formular absenden möchte
- eine neue Aufgabe/Aktion beginnen möchte

**Nicht eingesetzt** werden Buttons als navigatorische Elemente. Soll ein:e Nutzer:in auf eine neue Seite geleitet werden, wird stattdessen ein [Link](?path=/docs/components-navigation-link) verwendet.

## Varianten

Jede Button-Variante hat eine bestimmte Funktion, die durch das Design signalisiert wird. Daher ist es wichtig, dass die verschiedenen Varianten in Produkten konsistent eingesetzt werden, um Aktionen einheitlich zu kommunizieren.

Ein **Primary Button** wird für die wichtigste Aktion verwendet. Es soll nur ein Primary Button pro Screen verwendet werden (mit Ausnahme von Header, Dialogen oder Panels).

**Secondary Button**s werden für sekundäre Aktionen verwendet. Sie werden immer in Verbindung mit einem Primary Button verwendet und stellen die negative Aktion (z.B. "Abbrechen" oder "Zurück") dar. Secondary Buttons sollen nicht alleinstehend verwendet werden.

**Tertiary Button**s können für weniger prominente Aktionen verwendet werden. Sie können im Kombination mit einem Primary Button oder eigenständig eingesetzt werden.
