# Quote

Ein Quote wird genutzt, um Aussagen von Personen plakativ darzustellen und aus dem Fließtext hervorzuheben.

Ein Quote wird eingesetzt wenn:

- Ein besonders lesenswerter Satz aus dem Inhalt getrennt hervorgehoben werden soll
- Ein Satz auch beim skimmen der Website erfasst werden soll

## Anatomie

1. **Quote**: Hervorzuhebener Satz (ggf. verkürzt) aus dem Inhalt
2. **Autor**: Kann optional hinterlegt werden und optional mit einer Quelle verlinkt werden
