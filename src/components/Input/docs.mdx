# Input

Input-Felder erlauben den Nutzer:innen die Eingabe von Text. Sie tauchen in der Regel in Formularen oder Dialogen auf.

<Canvas>
  <Story id="components-input-input--without-value" />
</Canvas>

## Anatomie

Ein Input besteht aus dem eigentlichen Eingabefeld, einem Label und ein Platzhalter-Text (optional).

<!-- Optional: schematische Darstellung -->

1. **Label**: Beschreibt den einzugebenen Inhalt
2. **Input**: Eingabefeld für den Text des:der Nutzer:in
3. **Platzhalter** (optional): Kann verwendet werden um ein Beispielinhalt zu kommunizieren.
