# Video

Die Video-Komponente stellt einen Videoplayer dar.

Die Komponente **wird verwendet**, um:

- einen Videostream per HTML-Player abzuspielen
- ein Youtube-Video einzubinden
- ein Vimeo-Video einzubinden
