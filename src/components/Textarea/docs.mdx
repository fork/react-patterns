# Textarea

Eine Textarea ermöglicht Nutzer:innen die Eingabe von Text, der länger als eine Zeile ist.

<Canvas>
  <Story id="components-input-textarea--default" />
</Canvas>

## Anatomie

<!-- Optional: schematische Darstellung -->

1. **Label** (optional): Beschreibt den Inhalt der Textarea
2. **Placeholder** (optional): Hinweis, wenn Nutzer:innen noch keinen Text eingegeben haben
3. **Character Count** (optional): Stellt die Anzahl der eingegebenen und der maximal erlaubten Zeichen dar.
4. **Error**: Fehlermeldung
