# Tabs

Tabs ermöglichen Nutzer:innen, schnell zwischen verschiedenen Views innerhalb eines Kontextes zu wechseln.

Tabs **werden verwendet**, um unterschiedliche Inhalte auf der selben Ebene anzuordnen.

**Nicht verwendet** werden Tabs, um ähnliche Inhalte auf begrenztem Platz darzustellen. Stattdessen sollte ein [Accordion](?path=/docs/components-content-accordion) verwendet werden.

## Anatomie

Tabs bestehen aus einer horizontalen Gruppe von Schaltflächen zur Auswahl des Inhalts und einem Panel, dass den jeweiligen Inhalt darstellt.

<!-- Optional: schematische Darstellung -->

1. **Control**: Enthält das Label und macht als Schaltfläche das assoziierte Panel sichtbar.
2. **Panel**: Enthält die Inhalte des Tabs.
