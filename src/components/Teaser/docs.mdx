# Teaser

Ein Teaser gibt eine Vorschau auf einen verlinkten Inhalt und ermöglicht es Nutzer:innen, zu diesem Inhalt zu navigieren.

<Canvas>
  <Story id="components-navigation-teaser--default" />
</Canvas>

## Anatomie

<!-- Optional: schematische Darstellung -->

1. **Image** (optional): Bild als Vorschau auf den Zielinhalt
2. **Title**: Beschreibung des Zielinhalts
3. **Text** (optional): Kurzer Text, damit Nutzer:innen entscheiden können, ob sie zu dem Inhalt navigieren möchten
